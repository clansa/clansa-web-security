About Clansa Web Security

Origin story

I'm an 18-year-old who got into cybersecurity the hard way — jumping between YouTube videos, outdated blog posts, and courses that taught me enough to pass a quiz but not enough to actually understand anything. So I started writing my own notes. Not summaries, not cheat sheets — actual deep explanations of how things work at the protocol level. 800 pages later, Clansa Web Security exists.

What this site is

This site is built around one idea: most people learn cybersecurity at the surface level and wonder why nothing clicks. The free docs here cover the fundamentals that every other resource skips or oversimplifies — OSI, DNS internals, TCP, how browsers actually work. The Vault goes deeper, into the offensive side. No fluff, no hand-holding, no "cybersecurity in 10 minutes" energy.

Who it's for

If you're a complete beginner who wants to actually understand networking before touching a single tool — start with the free docs. If you already know the basics and want to learn how to break things properly — The Vault is for you. If you're looking for a certificate prep course, this isn't it.
